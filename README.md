Proyecto Consulta Médica

Descripción

Este proyecto consiste en el desarrollo de una aplicación backend para la gestión de consultas médicas. Su finalidad es facilitar el registro, seguimiento y gestión de pacientes, citas, médicos, y otros datos relacionados con una clínica.

Instalación

Clonar el repositorio:
git clone https://github.com/MileyLopezZz/Proyecto_Consulta_Medica.git

Instalar dependencias:
pip install -r requirements.txt

Aplicar migraciones:
py manage.py makemigrations y
py manage.py migrate

Ejecutar el servidor localmente:
py manage.py runserver

Abrir en el navegador la dirección http://127.0.0.1:8000/ para acceder al servicio.
